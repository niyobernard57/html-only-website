# NovaTech Solutions HTML-Only Website

## Overview
This is a **HTML-only website** for a fictional tech company called **NovaTech Solutions Ltd**.  
The website is designed to be **clean, readable, and professional**, showcasing the company’s services, media, and contact information.  

No JavaScript or frameworks are used — only **HTML5 and CSS3**.

---

## Project Structure

```
project-folder/
│── index.html
│── about.html
│── services.html
│── contact.html
│── form.html
│── media.html
│── css/
│   └── style.css
│── images/
│   └── devlab.jfif
    └── server.jfif
    └── workspace.png
│── media/
│   └── Audio.mp3
    └── Video.mp4  
    
```

- `index.html` — Homepage with company introduction and navigation  
- `about.html` — Company information and article with blockquote and address  
- `services.html` — List of services and a service table  
- `contact.html` — Contact details and embedded Google Map  
- `form.html` — Full HTML form for client requests (centered on page)  
- `media.html` — Image gallery, audio, and video  
- `css/style.css` — External CSS for styling the website

---

## Features

- **Fixed Header & Footer** on all pages  
- **Responsive layout** with media queries  
- **Professional form styling** with focus states  
- **Image gallery** with hover effects  
- **Audio and video embedding**  
- **Clean typography and color scheme**  
- **All HTML5 semantic elements used** (`<header>`, `<footer>`, `<main>`, `<section>`, etc.)  

---

## How to Use

1. Clone the repository:
```bash
git clone https://github.com/niyobernard57/html-only-website.git
```

2. Open `index.html` in any modern browser.

3. Navigate using the menu to explore all pages:  
   - Home → `index.html`  
   - About → `about.html`  
   - Services → `services.html`  
   - Contact → `contact.html`  
   - Form → `form.html`  
   - Media → `media.html`  

---

## Technologies Used

- **HTML5** — for semantic structure  
- **CSS3** — for styling, Flexbox, Grid, and responsive design  
- **No JavaScript** or external frameworks
